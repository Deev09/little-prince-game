# The Little Prince — Journey to the Rose

A 3D first-person game where **you are the Little Prince**, hopping from planet to planet
across a starfield, dodging falling asteroids on your way back to your rose.
Built with Three.js.

## 🎮 Game Features

- Full 3D first-person planet-hopping with spherical gravity — walk all the way around each tiny planet
- Six hand-built planets from the book: Asteroid B-612 (with volcanoes and a baobab), the King's planet,
  the Businessman's coins, the Lamplighter's lamp, the Geographer's planet, and the Rose planet
- Glowing asteroids streak through space with fiery particle trails — one hit and the journey ends
- A screen-space rose compass always points the way home
- Starfield with twinkling layers, nebulas, a sun, and bloom post-processing
- Ambient synth soundtrack and sound effects, all generated in-browser with WebAudio

## 🚀 Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm (comes with Node.js)

### Installation

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Open your browser and navigate to `http://localhost:1234`, then click to begin.

## 🎯 How to Play

- **W A S D** (or arrow keys) — move
- **Mouse** — look around
- **SPACE** — jump; press again mid-flight to soar toward wherever you're looking
- **SHIFT** — run
- **R** — drift back to your planet if you get lost in space
- Gravity pulls you to the nearest planet — leap the gap and the next world catches you
- Reach the glowing rose 🌹 without being struck by an asteroid

## 🛠️ Built With

- [Three.js](https://threejs.org/) — 3D rendering, with UnrealBloom post-processing
- [Parcel](https://parceljs.org/) — bundler
- Modern JavaScript (ES6+)
